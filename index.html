timegrid.io

take online reservations from your customers

The Problem

Small and medium scale businesses wish to provide online service booking, but tools out there are either too expensive or too complex.

The Solution

Timegrid is a single platform holding all businesses and customers while letting them an affordable and easy to use way to interact. Useful, Simple, Fast.

The Good News

Timegrid is open source, built with Laravel 5.1 framework for PHP, and you can find it on Github

This project is no longer supported. Codebase left public for experimental playground purpose.
